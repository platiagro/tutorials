Este é um componente que remove todos atributos com baixa variância usando uma implementação do Scikit-learn. Scikit-learn é uma biblioteca open source de machine learning que suporta apredizado supervisionado e não supervisionado. Também provê várias ferramentas para montagem de modelo, pré-processamento de dados, seleção e avaliação de modelos, e muitos outros utilitários.

Espera-se como entrada para o componente uma tabela com colunas que representam valores numéricos, categóricos ou de data. A tabela deve ser de um dos seguintes tipos: Comma-separated values (.csv) ou Excel (.xls, .xlsx).

A seguir são listados todos os parâmetros utilizados pelo componente:

  • Atributo alvo: feature.
    Esse valor será utilizado para garantir que o alvo não seja removido.
  • Limiar: float, padrão: 0.0.
    Atributos com variância menor que o limiar serão removidos.
